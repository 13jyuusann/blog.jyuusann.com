折腾两天终于又把博客建了起来，虽然这次也是不知道第几次重建博客，不说要一直坚持下去，先试着坚持域名买的这一年不断更吧。

名字

这个名字其实也没有什么特别的意思，很早之前给 QQ 小号随手打了个「12」的昵称，之后微信昵称就顺手打了个「13」，一直用到现在。

域名

熟悉日语的肯定一眼能看出来了，就是13的日语发音而已。

原因

毕业工作以后变得很浮躁，想要的太多又静不下心。想找个地方发些东西，但是各种 SNS 充斥着各种广告、限制、撕逼，实在是没法舒服地使用。正巧又看到了 十年之约 ，索性直接自己建个站想怎么玩怎么玩。

程序

这次用的是 Hexo + Github Pages，主题 Cards，想着与其折腾半天不如简单点，能在博客里多写点字比啥都强。评论暂时没有加上，用 SNS 账号带 Hashtag 发内容都没什么人看，这里应该就更难有人看到啦。

内容

以后博客主要就发发日记、其他自己想记录的东西、拍的照片，先争取每个月能至少写一篇吧。博客缺少的功能也比较多，需要些时间添加，顺便复习前端知识，自从换城市找工作没进前端行业后基本再也没摸过了。还是有一丢丢码农心，虽然发展成职业基本是没戏了，就当作爱好吧。

总之这是第一篇博文，以后加油吧 :)
